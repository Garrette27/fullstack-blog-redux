# Full Stack Blog - Redux 

A simple blog application built with React 19, TypeScript, Redux Toolkit, and Supabase.

## Features

- ✅ User Authentication (Register, Login, Logout)
- ✅ Blog CRUD Operations (Create, Read, Update, Delete)
- ✅ Image Upload for Blogs
- ✅ Blog Listing with Pagination
- ✅ Protected Routes
- ✅ TypeScript for Type Safety
- ✅ Redux Toolkit for State Management

## Tech Stack

- **Frontend**: React 19 + TypeScript
- **State Management**: Redux Toolkit
- **Backend**: Supabase (Auth + Database + Storage)
- **Routing**: React Router DOM
- **Build Tool**: Vite


### Global vs Local State
- **Global (Redux)**: Authentication, user data, blog list, pagination state
- **Local (useState)**: Form inputs, file selections, UI toggles, modal states
